Dataset: PTB-XL (PhysioNet)

Este proyecto usa el dataset PTB-XL disponible en:
https://physionet.org/content/ptb-xl/1.0.3/

Por razones de tamaño, este repositorio incluye solo una muestra:
    data/raw/records500/00000/

Estructura esperada del dataset completo:
    data/raw/
        ├─ records500/
        ├─ records100/
        ├─ ptbxl_database.csv
        └─ scp_statements.csv

Cada colaborador debe descargar el dataset completo desde PhysioNet
y colocarlo en la ruta correspondiente de su entorno local.
